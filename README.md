# Recon Detector v3

Recon automation tool for penetration testing.

## Features
- Crawler (HTML + JS + Param)
- Directory Scanner
- Subdomain Finder
- Port Scanner
- Endpoint Analysis

## Installation
git clone https://github.com/RenggaSenpaii/detectorv3.git
cd detectorv3
pip install -r requirements.txt

## Usage

### Method 1 (Executable)
chmod +x detectorv3
./detectorv3 -u target.com -a

### Method 2 (Python)
python3 detectorv3 -u target.com -a

## Modes
- basic
- web fingerprint
- sub
- dir
- crawl
- auto
- port
